# PRT(Peer Review Template)
- [ ]  **1. 주어진 문제를 해결하는 완성된 코드가 제출되었나요?**
    - 문제에서 요구하는 미션들을 모두 수행하였음
    - 문제점을 정확히 분석하여 해결하는 방법을 모색함

- [ ]  **2. 전체 코드에서 가장 핵심적이거나 가장 복잡하고 이해하기 어려운 부분에 작성된 
주석 또는 doc string을 보고 해당 코드가 잘 이해되었나요?**
    - 해당 코드의 기능, 존재 이유, 작동 원리 등을 기술하고 정확히 이해하고 있음
    - 주석을 보고 타인도 코드 이해가 잘 되도록 기록함
        
- [ ]  **3. 에러가 난 부분을 디버깅하여 문제를 해결한 기록을 남겼거나
새로운 시도 또는 추가 실험을 수행해봤나요?**
    - 문제 원인 및 해결 과정을 코드 과정에 기록되었음 
    - 추가적으로 수행한 본인만의 방식을 시도하여 실험을 기록 및 첨부하였음
    - 특히, 동물이 여러 마리 있는 사진을 통하여 여러 마리의 동물도 가능함을 보여주었음
        
- [ ]  **4. 회고를 잘 작성했나요?**
    - 주어진 문제를 해결하는 배운점과 아쉬운점을 정리하여 작성하였음
    - 추후 학습시 개선 및 발전 가능
        
- [ ]  **5. 코드가 간결하고 효율적인가요?**
    - 코드의 중복을 최소화하고 복잡성을 감소시킨 점이 특징적임


# 회고(참고 링크 및 코드 개선)
```
# 다양한 실험을 통하여 코드 방식을 본인만의 방식으로 시도하였다는 점에서 배울 점이 많음
```
